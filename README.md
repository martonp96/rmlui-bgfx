# rmlui-bgfx

## Build
- Install [xmake](http://xmake.io)
- Run the xmake command

## Generate Visual Studio project files
- Run generate_project.bat

## Platform compatibility (tested)
|                        | D3D11 |  Vulkan | Metal |
|------------------------|-------|---------|-------|
| Windows 11             |  ✅   |    -    |   -   |
| Ubuntu 22.04 (Wayland) |   -   |    ✅   |   -   |
| macOS                  |   -   |    -    |   ✅  |
